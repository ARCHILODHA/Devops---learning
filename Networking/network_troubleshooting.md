# Network Troubleshooting Commands

## Check IP

```bash
ip addr
```

## Ping

```bash
ping google.com
```

## DNS Lookup

```bash
nslookup google.com
```

## Trace Route

```bash
traceroute google.com
```

## Active Connections

```bash
netstat -tuln
```

## Check Routes

```bash
ip route
```

## Summary

These commands help diagnose:
- Connectivity issues
- DNS failures
- Routing problems
- Open ports
- Interface configuration
