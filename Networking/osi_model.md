# OSI Model

## Overview
The OSI (Open Systems Interconnection) Model is a 7-layer framework used to understand network communication.

| Layer | Name | Example |
|-------|------|---------|
| 7 | Application | HTTP, FTP, DNS |
| 6 | Presentation | SSL/TLS, JPEG |
| 5 | Session | NetBIOS |
| 4 | Transport | TCP, UDP |
| 3 | Network | IP, ICMP |
| 2 | Data Link | Ethernet, MAC |
| 1 | Physical | Cables, Fiber |

## Mnemonic
All People Seem To Need Data Processing

## Why It Matters
- Helps troubleshoot networking problems.
- Standardizes communication.
