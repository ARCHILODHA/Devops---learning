# Subnetting Basics

## What is Subnetting?

Subnetting divides a network into smaller subnetworks.

Example:

192.168.1.0/24

Subnet Mask:

255.255.255.0

## Benefits

- Better security
- Reduced broadcast traffic
- Efficient IP allocation

## CIDR Examples

/24 = 256 IPs
/25 = 128 IPs
/26 = 64 IPs
